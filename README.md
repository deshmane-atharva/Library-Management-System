The Library Management System is a web-based application developed to efficiently manage the daily operations of a college library. This project is built using Python (Flask Framework) for backend development, HTML and CSS for frontend design, and Supabase as the cloud database for secure and reliable data storage.

The system provides a user-friendly interface that allows librarians to manage books, members, and transactions easily. It helps in automating manual library tasks such as issuing books, returning books, managing member details, and tracking transaction history.
